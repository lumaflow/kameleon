Getting Color Right

Why should we care?

Cynthia Brewer

“Appropriate use of color for data display allows interrelationships and patterns within data to be easily observed. The careless use of color will obscure these patterns.”

Color Use Guidelines for Mapping and Visualization

Stephen Few

“Don’t use color to decorate the display. Dressing up a graph might serve a purpose in advertising, but it only distracts people from what’s important—the data—in an information display.”

Practical Rules for Using Color in Charts

Maureen Stone

“Color used well can enhance and clarify a presentation. Color used poorly will obscure, muddle and confuse…using color well in information display is essentially about function.”

Choosing Colors for Data Visualization
